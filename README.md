# Basic Chatbot using Python

## Description

This project is developed as part of the CodeAlpha Python Programming Internship.

It is a simple rule-based chatbot that responds to predefined user inputs such as greetings, questions, and exit commands.

## Features

* Greets the user
* Answers predefined questions
* Runs continuously until the user types "bye"
* Beginner-friendly implementation

## Technologies Used

* Python 3

## How to Run

```
python chatbot.py
```

## Sample Commands

* hello
* how are you
* what is your name
* help
* bye

## Learning Outcomes

* Functions
* Loops
* Conditional Statements
* User Input
* Basic Chatbot Logic
